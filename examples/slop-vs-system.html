Slop vs System

Every pair below exposes one rule. The left side is a common AI-slop default. The right side is the system-compliant rendering. Read the rule ID, feel the difference, recognize the pattern next time an agent tries to ship the left.

✗ Slop
TY01Primary face is Inter
Elite, industry-defining craft
Inter is the default AI-generation fingerprint. It reads as "generic product UI" because it is the generic product UI. Reject on sight.
✓ System
TY01Primary face is Geist
Elite, industry-defining craft
Geist is geometric, neutral, and characterful without tipping into personality. Free. Self-hostable. Fallback stack has matching x-height to prevent FOUT shift.
✗ Slop
EL02box-shadow blur ≥ 25px

Card

Drops a giant soft shadow onto a dark canvas. This is the AI-slop signature.

On dark canvas, a large soft drop shadow reads as "floating UI theater." Large blur = amateur signal. The linter flags any shadow with blur ≥ 25px.
✓ System
EL021px border, no shadow

Card

Same message, separated by a 1px border. Edge is crisp. No glow.

Surface hierarchy via 6-point luminance steps + 1px border. No elevation theater. Shadows reserved for menus, toasts, and modals only.
✗ Slop

The red-orange accent now means "buy this." It also means "live, now, active" elsewhere in the UI. Two meanings, one color = meaning collapses.
✓ System
CL04primary text color as button bg
The CTA uses luminance contrast, not hue. Accent stays reserved for temporal semantics. One accent, one meaning.
