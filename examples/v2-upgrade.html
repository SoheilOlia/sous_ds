PAGE — RECIPES

Six named archetypes.

Each recipe declares its intent, primary primitive, microcopy template, and forbidden substitute. A page must use ≥3 distinct recipes when the rhythm dial is at default. No more card-grid monotony.

PipelineMap

Multi-stage process with order and per-stage state.

primary: <DotTimeline> + <DottedChart>
MilestoneStrip

Time-tagged phases with done / active / queued state.

primary: <DotTimeline> + <SegmentedBar>
AgentLog

Live agent or tool activity with a "now" head.

primary: <ToolCall> + <PulseTrail>
ReceiptStack

Completed events as machine-attested truth.

primary: <ToolCall tone="done">
MetricWall

2–4 numbers sharing a unit or axis.

primary: <MetricStat> row
RAGStatus

Single state-word callout (RED / AMBER / GREEN).

primary: <LiveBlock> + <Pill>
